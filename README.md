# Integrated Sales & Business Operations Performance Dashboard

## Project Overview
This project is a Data Analytics and Visualization project created using Tableau.

It analyzes sales, profit, customers, products, and geographical performance.

## Tools Used
- Tableau Desktop
- Tableau Public
- Microsoft Excel (data source)

## Dataset
Classic Models Dataset (provided by my mentor)

## Key Features
- 8 Worksheets
- 4 Interactive Dashboards
- Calculated Profit Field
- Filters and Dashboard Actions
